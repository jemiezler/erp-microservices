# HR Frontend Configuration

## Environment Variables

Create a `.env.local` file in the hr-mfe directory:

```env
# HR Service API
NEXT_PUBLIC_HR_API_URL=http://localhost:8081/api/v1

# Authentication
NEXT_PUBLIC_AUTH_URL=http://localhost:8080/api/v1/auth

# Multi-tenancy
NEXT_PUBLIC_TENANT_ID=default-tenant
```
